Full ranking Back to top

Full rankings of a season/division, sorted by position.

<div data-ranking=""
    data-season="1516"
    data-province="national"
    data-division="1"></div>
data-season

Season code: 1516 for example.

data-province

Province code (shorthand) or "national"

data-division

Division code.

Examples:

  • Brabant - A-squad 2015-2016
  • National - RSC Anderlecht 2015-2016 
<div data-ranking=""
        data-season="1516"
        data-province="bra"
        data-division="3D"></div>
<div data-ranking=""
        data-season="1516"
        data-province="national"
        data-division="1"></div>

Short ranking Back to top

Short rankings to display as a widget. Will retrieve the ranking of the team (in division/season we define), and fetch a defined number of teams who are ranked above and below. If there are no (or too few) teams ranked higher/lower, the focus will shift to the other side (so for example if the team we search for is in first position and we want 5 results, we will display our team + 4 teams below it).

<div data-ranking_short=""
    data-season="[season]"
    data-province="[province code]"
    data-division="[division code]"
    data-teamname="[team name]"
    data-number="[number of total results]"></div>
data-season

Season code: 1516 for example.

data-province

Province code (shorthand) or "national"

data-division

Division code.

data-teamname

Teamname, or part of it, to highlight and focus our short ranking on.

data-number

Number of results to display in total. Odd number is preferred.

Examples:

  • Brabant - A-squad 2015-2016
  • National - RSC Anderlecht 2015-2016 
<div data-ranking_short=""
        data-season="1516"
        data-province="bra"
        data-division="3D"
        data-teamname="KCVV.Elewijt"
        data-number="5"></div>
<div data-ranking_short=""
        data-season="1516"
        data-province="national"
        data-division="1"
        data-teamname="R.SC.Anderlecht"
        data-number="7"></div>

Matches overview Back to top

Create a listing of all matches, grouped per matchday, or select an individual matchday to display.

<div data-matches_overview=""
    data-season="[season]"
    data-province="[province code]"
    data-division="[division code]"
    data-logo="[display teamlogos]"
    data-matchday="[specific matchday or 0]"></div>
data-season

Season code: 1516 for example.

data-province

Province code (shorthand) or "national"

data-division

Division code.

data-matchday

Set to 0 for all matches, otherwise pick a specific matchday.

[data-logo]

Optional. Set to TRUE if you want to display club logos in the match overview.

Examples:

  • Brabant - A-squad 2015-2016
  • Brabant - Cup 2015-2016 
<div data-matches_overview=""
        data-season="1516"
        data-province="bra"
        data-division="3D"
        data-matchday="5"
        data-logo="TRUE"></div>
<div data-matches_overview=""
        data-season="1516"
        data-province="bra"
        data-division="BCF"
        data-matchday="0"></div>
